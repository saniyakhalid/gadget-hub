# MERN E-Commerce Application 🛒

A full-stack **E-Commerce Web Application** built using the **MERN stack** (MongoDB, Express, React, Node.js).  
The application supports **user authentication using JWT**, **email services using Nodemailer**, and **secure online payments using Razorpay**.

---

## 🚀 Features

### 👤 User Features
- User registration and login
- JWT-based authentication and authorization
- Secure password handling
- Email notifications using Nodemailer
- Browse products
- Add products to cart
- Place orders and make payments using Razorpay

### 🛍️ Product & Order Features
- Product listing with details
- Shopping cart functionality
- Order creation and management
- Secure online payment integration (Razorpay)
- Order confirmation via email

### 🔐 Security Features
- JSON Web Token (JWT) authentication
- Protected routes for authorized users
- Secure payment handling through Razorpay
